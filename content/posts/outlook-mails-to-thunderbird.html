I had some outlook backup files and I wanted to open them in Thunderbird. I spent some time figuring out how to do it. So, documenting it here. I know there might be better ways to do it, but it these steps worked for me.

  1. Convert PST file to MBOX using PST converter tool like Sysinfo Linux PST Converter
  2. Create a local folder (xxx) and add it to Thunderbird (Add LocalFolders Extension if needed) and close thunderbird.
  3. Navigate to the folder you created (xxx), and you can see a list of files for each mail folder (e.g., Inbox and Inbox.msf).
  4. Rename the MBOX file you created to one of the file names (eg., Inbox) and copy to the above folder, overwriting the Inbox file (leave Inbox.msf untouched).
  5. Open Thunderbird, and you should be able to read your Outlook mails.

If you have multiple .pst files, you might need to repeat these steps for each file.
